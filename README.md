# 📊 지역별 인구 구조 분석

행정안전부의 연령별 인구현황 데이터를 활용하여 지역별 인구구조를 시각화하고, 선택한 지역과 인구구조가 가장 비슷한 전국 지역을 찾아주는 Streamlit 웹 애플리케이션입니다.

시도·시군구뿐 아니라 읍면동 단위의 행정구역도 검색하고 비교할 수 있습니다.

---

## 주요 기능

### 1. 지역별 연령 인구 시각화

사용자가 선택한 지역의 연령별 인구수를 Plotly 꺾은선 그래프로 확인할 수 있습니다.

* 기준 연월 선택
* 복수 지역 선택
* 전체·남성·여성 선택
* 0세부터 100세 이상까지 연령별 인구 표시
* 마우스 오버 시 나이와 인구수 확인

### 2. 지역 총인구수 요약

선택한 각 지역의 총인구수를 요약 지표로 표시합니다.

### 3. 인구구조 유사 지역 Top 5

선택한 기준 지역과 전국 행정구역의 연령별 인구구조를 비교하여 가장 비슷한 지역 5곳을 찾습니다.

인구 규모가 다른 지역도 비교할 수 있도록 절대 인구수가 아닌 각 연령 인구의 비율을 사용합니다.

### 4. 유사도 시각화

유사 지역 검색 결과는 다음과 같이 제공됩니다.

* 코사인 유사도 점수 막대그래프
* 기준 지역과 유사 지역의 연령별 인구 비율 비교
* 유사도와 총인구수가 포함된 상세 표

---

## 유사도 계산 방식

각 지역의 연령별 인구수를 총인구수로 나누어 연령별 인구 비율 벡터를 생성합니다.

```text
연령별 인구 비율 = 해당 연령 인구수 / 지역 총인구수
```

두 지역의 인구구조 유사도는 코사인 유사도를 사용하여 계산합니다.

```text
cosine_similarity(A, B) = (A · B) / (||A|| × ||B||)
```

유사도 점수는 `1`에 가까울수록 두 지역의 연령별 인구구조가 비슷하다는 의미입니다.

기준 지역 자체는 검색 결과에서 제외됩니다.

## 데이터 전처리

행정안전부 CSV 데이터는 다음 과정을 거쳐 처리됩니다.

### 행정구역 코드 제거

원본 행정구역 값 뒤에 포함된 숫자 코드를 제거하여 화면에 표시할 지역명을 생성합니다.

```text
서울특별시 종로구 청운효자동 (1111051500)
```

위 값은 다음과 같이 표시됩니다.

```text
서울특별시 종로구 청운효자동
```

### 숫자 컬럼 변환

CSV의 인구수 값은 쉼표가 포함된 문자열 형태일 수 있습니다.

```text
"1,234"
```

총인구수와 연령별 인구수를 포함한 모든 값 컬럼에서 쉼표를 제거한 뒤 숫자형으로 변환합니다.

```python
value_cols = [
    column
    for column in df.columns
    if column not in ("행정구역", "지역명")
]

for column in value_cols:
    df[column] = pd.to_numeric(
        df[column]
        .astype(str)
        .str.replace(",", "", regex=False),
        errors="coerce",
    )
```

이 과정을 통해 다음과 같은 숫자 포맷을 정상적으로 사용할 수 있습니다.

```python
f"{total_pop:,.0f}명"
```

## 사용 방법

1. 사이드바에서 기준 연월을 선택합니다.
2. 분석할 지역을 하나 이상 선택합니다.

6. 인구구조가 가장 비슷한 전국 지역 Top 5를 확인합니다.

지역 선택창에 지역명을 직접 입력하여 읍면동 단위까지 검색할 수 있습니다.

## 오류 해결 기록

### 총인구수 포맷 오류

다음 코드에서 오류가 발생할 수 있었습니다.

```python
f"{total_pop:,.0f}명"
```

원인은 총인구수 값이 `"1,234"`와 같은 문자열로 남아 있었기 때문입니다.

초기 코드에서는 연령별 인구 컬럼만 숫자형으로 변환했지만, 수정된 코드에서는 행정구역 관련 컬럼을 제외한 모든 데이터 컬럼을 일괄적으로 숫자형으로 변환합니다.

### CSV 파일을 찾을 수 없는 경우

다음 오류 메시지가 표시될 수 있습니다.

```text
CSV 파일을 찾을 수 없습니다.
```

다음 항목을 확인합니다.

* CSV 파일이 `app.py`와 같은 폴더에 있는지 확인
* 실제 CSV 파일명과 `CSV_PATH`가 일치하는지 확인
* GitHub 저장소에 CSV 파일이 정상적으로 업로드되었는지 확인

### 한글 인코딩 오류

행정안전부 CSV 파일은 일반적으로 `cp949` 인코딩을 사용합니다.

```python
pd.read_csv(path, encoding="cp949")
```

환경이나 파일에 따라 인코딩이 다른 경우 `utf-8-sig` 등을 시도할 수 있습니다.

## 참고 사항

* 인구가 0명인 지역은 인구 비율을 계산할 수 없어 유사도 비교에서 제외됩니다.
* 데이터가 없는 지역은 그래프 또는 검색 결과에 나타나지 않을 수 있습니다.
* 현재 유사도 비교는 전체 성별인 `계` 데이터를 기준으로 계산합니다.
* 연령별 구조 비교는 0세부터 100세 이상까지의 데이터를 사용합니다.
* 데이터 파일이 클 경우 최초 로딩과 유사도 계산에 시간이 걸릴 수 있습니다.
* Streamlit의 캐시 기능을 사용하여 반복 실행 시 데이터 처리 속도를 개선합니다.
